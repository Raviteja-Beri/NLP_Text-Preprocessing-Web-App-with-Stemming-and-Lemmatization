# Text Preprocessing Web App with Stemming and Lemmatization

This repository contains a **Natural Language Processing (NLP)** application built with **Python**, **Flask**, and **NLTK**.  
It provides multiple text processing functionalities such as **tokenization**, **stemming**, and **lemmatization**, accessible via both a **web interface** and a **command-line tool**.

---

## üöÄ Features

- **Sentence Tokenization** ‚Üí Splits text into sentences.
- **Word Tokenization** ‚Üí Splits text into words.
- **WordPunct Tokenization** ‚Üí Splits text into words & punctuation separately.
- **Stemming** ‚Üí Reduces words to their root forms using **Porter Stemmer**.
- **Lemmatization** ‚Üí Converts words to base form using **WordNet Lemmatizer**.
- **Stopword Removal** ‚Üí Removes common English stopwords for cleaner results.
- **Flask Web Interface** ‚Üí Interactive UI for entering text and choosing processing method.
- **CLI Script (`tokening.py`)** ‚Üí Direct terminal-based execution for NLP tasks.
